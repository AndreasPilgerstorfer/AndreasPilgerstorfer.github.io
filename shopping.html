Covid - Social Shopping

In this project I have created a responsive web application using object oriented JavaScript/jQuery,with the MVC pattern, HTML and CSS. The aim of this application is to help persons which aren't able to do their own shopping anymore due to the Covid-19 crisis. In th app, there are two different roles available: help seekers and help providers. Help seekers can create lists and add articles to the list. Helpers can choose a list from the seekers and take them in order to buy the desired articles on the list. By entering the final price, they are able to fulfill the list completely.

Disclaimer: If you want to try the application you have the possibility to view the whole functionality if you dont click one of the buttons for the desired role. If you click on one of the buttons you can experience the app from the different perspectives.
The app is offered in German.
