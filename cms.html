Typo 3 Website

In this project I have created a website for a fictional sports club by the name of SV Kreuzstadt. The website is created by using the content management Typo3. It includes features like a news section, a gallery, and a special area for registered users. The entire website is translated in English and German.


The website is available in German and English.
