UX/UI app prototype

This project is about a dynamic app prototype which is based on different usability design evaluations and interaction design principles. The prototype was created in a team project using the tool Axure RP9. The prototype is about an app, PETsi, which allows users to find the perfect non-human soulmate in a pet by using a specific test. It also allows users to manage all information about their already existing pets. We won the first place out of eight participating groups at the usability challenge at the university of applied sciences upper austria with this prototype.


The video is only available in German.
